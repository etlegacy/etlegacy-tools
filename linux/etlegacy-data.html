ET: Legacy requires data

Even when ET: Legacy is a free software and open-source, it requires additional data files from original Wolfenstein: Enemy Territory. Without the game data this game cannot work.

Data were not licensed under free software license, which is a reason we could not ship them to you. Do not worry, Splash Damage still provides original game including required data free of charge for non-commercial use.

How to install data

  1. Download old from Splash Damage
  2. Use game-data-packager
  3. Install etlegacy-installer and start it from menu

Download from Splash Damage

  1. Open Wolfenstein: Enemy Territory
  2. Choose Download, Linux Full game
  3. unzip the archive
  4. Open the directory in terminal
  5. sudo sh et260b.x86_keygen_V03.run
  6. Skip etkey creation
  7. Read and accept the license
  8. Use default paths
  9. Choose no to install startup entries

Install data using game-data-packager

  1. Install package dame-data-packager
  2. run game-data-packager enemy-territory -i
  3. Ensure you understand and accept the license
